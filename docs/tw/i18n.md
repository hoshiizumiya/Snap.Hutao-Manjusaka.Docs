# 國際化支持
::: warning
Manjusaka 版本的 Snap Hutao 目前處於維護階段，國際化支持或優化請主動提出相關 issue，部分語言的翻譯效果不良或者內容過時，敬請諒解。
:::

Snap Hutao 以支持原神國服功能為主，目前已逐步完善對國際服的支持，涵蓋語言本地化和功能國際化。
由於兩種服務器均有不同類型的限制，故在部分功能上的實現可能會有所不同，如國際服的祈願記錄導出不支持更加方便的 SToken 刷新方式。
