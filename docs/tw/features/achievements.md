# 成就管理

胡桃工具箱提供全面的原神成就管理功能，支持瀏覽與管理所有成就狀態。用戶可以通過多種篩選方式快速定位成就，優化管理體驗。

<!-- more -->

![成就管理界面](https://img.alicdn.com/imgextra/i1/1797064093/O1CN01fApvim1g6e0xyGQvS_!!1797064093.png_.webp)

首次使用時，根據提示點擊“創建新存檔”按鈕以創建一個成就存檔併為其命名。創建第一個存檔後，用戶可以通過點擊右上角的“創建新存檔”按鈕，為不同的賬號創建獨立的成就檔案。

## 成就篩選

胡桃工具箱支持多種靈活的成就篩選方式：

- 點擊右上角隱藏菜單，選擇 `僅委託成就` 選項，篩選出需要通過每日委託完成的隱藏成就。
- 在搜索框中輸入版本號（例如 `5.3`）以查找對應版本的成就。
- 輸入相關怪物名稱（例如 `無相之火`）快速定位相關成就。
- 輸入成就的部分名稱（例如 `四風`）進行模糊搜索。
- 輸入完整的成就編號（例如 `84575`）精準查找目標成就。

## 成就導入 <Badge text="UIAF" type="info" />

胡桃工具箱支持通過 [UIAF 統一可交換成就標準](https://uigf.org/zh/standards/uiaf.html) 導入成就數據。

- 用戶可以通過 URL 協議或剪貼板，從**第三方成就導出工具**中導入數據。
- 點擊成就頁面右上角隱藏菜單中的“導入”按鈕，選擇“從 UIAF 文件導入”，即可加載符合 UIAF 數據格式的成就數據。

::: tip
如果用戶安裝了多個版本的胡桃工具箱（如 Alpha 版和正式版），需要更改 `URL:hutao` 鏈接的默認指向應用，可在“Windows 設置-應用-默認應用-按協議指定默認應用”中找到 `URL:hutao` 項，點擊圖標進行更改。
:::

## 成就導出 <Badge text="UIAF" type="info" />

點擊成就頁面右上角隱藏菜單中的“導出”按鈕，選擇目錄並設置文件名後，成就數據將以 UIAF 格式導出並保存到指定位置。

## 刪除存檔

用戶可通過右上角隱藏菜單中的“刪除當前存檔”選項，刪除當前成就存檔。
