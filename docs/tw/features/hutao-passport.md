# 胡桃賬號

胡桃賬號是由 DGP-Studio 團隊創建的為胡桃工具箱服務的賬號系統，以使用在包括胡桃雲祈願記錄服務等相關的涉及雲端服務的功能。你可以在 Snap Hutao 客戶端左側菜單欄中點擊進入胡桃通行證頁面，查看和管理你的胡桃賬號。

## 雲服務說明

因雲相關的功能消耗了大量服務資源，因此有時間限制。目前，胡桃提供以下幾種啟用雲同步功能的方法：

1. 在愛發電上打賞胡桃的用戶，我們會向與愛發電郵箱賬號相匹配的胡桃賬號自動增加使用時間作為胡桃工具箱開發團隊對打賞用戶的感謝
   - 付款後您應當收到一份郵件回執，您可以點擊胡桃賬號右側的“刷新”按鈕以刷新續期狀態。因數據同步存在延遲，如刷新失敗，請等待約5分鐘再次嘗試刷新。
2. 對於所有註冊用戶，在**每一期**深淵每個原神賬號**第一次**成功向胡桃數據庫上傳深境螺旋挑戰記錄後，會獲得 3 天的祈願同步功能使用時間
   - 你可以通過該方法長期地免費使用胡桃雲祈願記錄服務
3. **為胡桃工具箱開源社區貢獻（包括但不限於代碼開發、程序設計、本地化翻譯）的用戶，可以直接聯繫開發團隊獲取開發者許可證或兌換碼**

## 胡桃雲祈願記錄服務

::: warning
當你的胡桃賬號雲同步權限過期後，你將無法上傳、下載和刪除數據，但你的數據不會被刪除。你可以在恢復權限後重新操作你的雲備份數據。
:::

如果你有胡桃雲權限，你將獲得如下權限：

1. 祈願記錄頁面中的“預測”功能
2. 祈願記錄的雲端同步功能

::: info 胡桃賬號與原神賬號對應關係
注：原神賬號的數據與胡桃賬號是“**多對一**”的關係。即，一個胡桃賬號可以關聯多個原神賬號的祈願數據，而一個原神賬號只能與一個胡桃賬戶關聯。

如果您因為購買了二手賬號等相關原因無法上傳祈願記錄，為數據安全考慮，請您與前號主協商解決，胡桃工具箱不會、也無法解決相關祈願衝突的問題。
:::

## 胡桃雲 CDN 服務

本功能允許用戶通過在中國境內部署的服務器下載胡桃的更新程序。通常情況下，使用此方式可以大幅提升中國大陸用戶的下載速度。

## 修改郵箱/修改密碼/註銷賬號/刷新賬號狀態

根據相關需求點擊相關按鈕即可。請注意，修改密碼、修改郵箱後將自動登錄修改後的賬號。

如果您在“愛發電”等網站贊助了胡桃但賬號顯示狀態未改變，您可以點擊賬號右側的“刷新”按鈕手動刷新狀態。若等待 5 分鐘後手動刷新狀態仍未正確顯示，請聯繫開發人員。
