# 我的角色

::: info 前置要求
使用此功能需登錄米遊社或 HoYoLAB 賬號，具體登錄方法請參見[米遊社多賬號切換](mhy-account-switch.md#米遊社多賬號切換)。
:::

“我的角色”功能能夠讀取用戶當前賬號中所有角色的詳細數據，包括等級、命之座、武器、天賦、聖遺物等內容，助力玩家優化角色培養與管理。

![網格視圖示意圖](/images/202501/characters_data.webp)

## 同步角色信息

點擊右上角的“同步角色信息”按鈕即可更新以下數據：

- 角色等級與角色命座
- 好感度
- 武器等級與精煉度
- 天賦等級
- 衣裝信息
- 聖遺物及其詳細屬性

## 將當前選定角色加入養成計算

1. 確保已同步角色數據。
2. 在“我的角色”功能頁面選擇需要加入養成計算的角色。
3. 點擊右上方的“養成計算-當前角色與武器”按鈕。

- 系統會自動預填同步過的角色等級、天賦等級及武器等級，用戶可自由修改目標等級。

![培養示例圖片](/images/202501/character_develop.webp)

完成添加後，可在“養成計劃”功能頁面查看已設定的養成目標。

詳細功能說明請參閱[養成計劃](./develop-plan.md#養成計劃)文檔。

## 導出角色詳情至剪貼板

“我的角色”頁面支持將角色數據以文本形式導出：

- 選擇目標角色後，點擊右上角的“導出文本到剪貼板”按鈕。
- 導出的數據存儲於系統剪貼板，可直接粘貼分享。
