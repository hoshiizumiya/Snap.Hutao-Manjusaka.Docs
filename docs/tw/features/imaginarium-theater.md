# 幻想真境劇詩

胡桃工具箱的“幻想真境劇詩”功能致力於為玩家提供該挑戰模式的數據記錄與統計服務。通過該功能，玩家可以歸檔自己的挑戰數據，並參與全服數據統計分析。

該功能主要分為以下兩個部分：

- **分期詳情**：展示玩家個人的幻想真境劇詩挑戰數據，包括詳細的層級進度、隊伍配置、通關記錄等。
- **本期統計**：展示上傳至胡桃數據庫的所有玩家挑戰時產生的角色出場數據

## 查看個人挑戰數據 （分期詳情）

- 點擊左側菜單欄進入“幻想真境劇詩”頁面，選擇“分期詳情”標籤。
- 查看本期和往期的挑戰記錄，包括：
  - 開幕角色陣容，特邀角色，全部待命角色，最強一擊，最佳紀錄等
  - 每一幕的怪物陣容，出戰角色陣容，輝彩祝福和神秘收穫

### 數據歸檔

- 胡桃工具箱會自動為每期挑戰歸檔，玩家可以查看往期挑戰數據，方便回顧歷史表現與調整策略。
- 點擊“刷新數據”按鈕以獲取最新的遊戲記錄。

## 全球挑戰數據（分期統計）

::: warning 數據範圍
所有統計數據均基於玩家自行上傳的記錄，可能無法完全代表所有玩家。
:::

- 點擊左側菜單欄進入“幻想真境劇詩”頁面，選擇“本期統計”標籤。
- 查看本期挑戰的角色出場信息

### 上傳與共享數據

1. 在“幻想真境劇詩”頁面中點擊“上傳數據”按鈕。
2. 系統會自動解析最新的挑戰記錄並提交至胡桃數據庫。
3. 上傳完成後，玩家會收到確認提示。
