# 武器資料

![weapon-wiki](https://img.alicdn.com/imgextra/i2/1797064093/O1CN018rs0K91g6e0ylQ4XT_!!1797064093.png_.webp)

武器資料是胡桃工具箱中內置的 WIKI 功能，專為查詢原神中所有武器的**外觀、參數、精煉效果、推薦搭配**等信息而設計。通過該功能，玩家可以高效地選擇適合角色的武器，並獲取詳細的武器數據支持。

## 已包含的數據

- **武器列表**

  - 列表支持兩種顯示方式，可通過左上角的按鈕切換：
    - **列表視圖**：顯示武器縮略圖、名稱及部分信息。
    - **網格視圖**：以縮略圖方式展示更多武器項。
  - 在“列表視圖”下，單擊縮略圖或名稱即可跳轉至武器的詳細介紹頁面。

- **武器基本信息**

  - 突破前後的武器圖標對比，便於識別升級後的視覺變化。
  - 立繪展示，顯示武器的完整外觀。
  - 武器簡介，提供武器的背景故事或相關描述。
  - 支持切換至對應等級，查看武器各項數值（如基礎攻擊力、副屬性等）。

- **武器精煉信息**

  - 展示每個精煉階數的具體屬性增益，便於玩家評估精煉價值。

- **武器養成材料**

  - 列出升級和突破所需的材料及數量。

- **武器使用推薦**
  - 基於胡桃數據庫統計信息，展示推薦的武器搭配角色和使用場景，幫助玩家優化武器選擇。
