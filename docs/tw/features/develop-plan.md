# 養成計劃

## 添加養成目標進入材料清單

### 基於 Wiki 資料添加

1. 在胡桃工具箱的左側欄目中，點擊“角色資料”或“武器資料”功能。
2. 點擊界面右上角的“養成計算”按鈕。
3. 在彈出的“添加或更新到當前養成計劃”窗口中，填寫角色或武器的起始等級與目標等級。

- 基於 Wiki 資料添加的養成目標，初始等級默認為 1 級（可根據需求調整）。

### 基於玩家數據添加

1. 在左側欄目中，點擊“我的角色”功能頁面，選擇需要加入養成計劃的角色。
2. 點擊界面右上角的“養成計算-當前角色與武器”按鈕。
3. 在彈出的“添加或更新到當前養成計劃”窗口中，填寫角色的起始等級與目標等級。

- 基於玩家數據添加的養成目標，默認初始等級為角色當前等級。
- 詳細信息請參見[我的角色](character-data.md#將當前選定角色加入養成計算)文檔。

## 新建計劃並同步遊戲內揹包物品

1. 在左側欄目中，點擊“養成計劃”功能。
2. 點擊右上角“新建計劃”併為其命名。
3. 按照[將當前選定角色加入養成計算](./character-data.md#將當前選定角色加入養成計算)或[將 Wiki 中角色添加至養成計劃](./character-wiki.md#將-wiki-中角色添加至養成計劃)的指引添加角色到新計劃。
   - 至少添加一個等級或武器等級未滿的角色，滿級角色無法添加至養成計劃。
4. 切換至“揹包物品”頁籤並點擊“同步揹包物品”按鈕。
   - 系統將自動同步當前登錄賬號的遊戲揹包物品。
   - 可通過“同步揹包物品”按鈕左側的下拉選項框為不同賬號選擇不同養成計劃。
   - 點擊任意物品圖標或數字區域即可手動編輯該材料數量。
   - 修改後的物品信息將自動關聯到所選的養成計劃。
5. 在“材料統計”頁面查看素材刷取的預估天數。

![體力預估功能](/images/202501/character_develop_predict.webp)

::: tip
若物品名稱與圖像顯示不一致或出現堆疊等顯示問題，可切換頁面後再返回本頁面解決。此問題由微軟導致，目前無法完全修復。
:::

## 刪除養成計劃

在胡桃工具箱的界面右上部分，點擊“刪除計劃”按鈕刪除當前養成計劃。刪除後，該計劃中的角色信息、揹包物品信息等數據將一併被清除。
