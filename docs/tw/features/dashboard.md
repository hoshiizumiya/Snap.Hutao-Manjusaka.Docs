# 首頁儀表盤

![儀表盤](https://img.alicdn.com/imgextra/i2/1797064093/O1CN01Nlz8ca1g6e0tyxrBa_!!1797064093.png_.webp)

啟動胡桃工具箱後，默認進入首頁儀表盤。儀表盤分為上下兩部分：上半部分以卡片的形式顯示個人遊戲數據概覽，下半部分展示近期遊戲公告及活動倒計時。主頁卡片可在設置中關閉或重新啟用。

## 功能概覽

- **Snap Hutao 公告**

  - 若開發團隊發佈新公告，將在儀表盤頂部顯示，大部分公告支持關閉。

- **遊戲啟動器**

  - 用戶可直接在儀表盤中選擇賬號並啟動遊戲。
    - 若啟用了“使用米遊社 / HoYoLAB 登錄”選項，賬號選擇功能將不可用。

- **祈願記錄**

  - 顯示當前祈願檔案的 UID，並以進度條形式呈現保底計數。
  - 對於擁有多個祈願檔案的用戶，可通過祈願記錄區域兩側的箭頭按鈕切換檔案。

- **成就統計**

  - 顯示成就檔案的已完成成就總數、完成百分比及最新獲得的成就。
  - 多個成就檔案的用戶，可通過成就統計區域兩側的箭頭按鈕切換檔案。

- **實時便箋**

  - 提供實時便箋數據的快速訪問。
  - 多個實時便箋的用戶，可通過實時便箋區域兩側的箭頭按鈕切換檔案。

- **日曆功能**

  - 顯示當天過生日的角色及當前可刷取的天賦、角色、武器材料。
    - 若“養成計劃”中包含當日可刷取的材料，相關圖標將突出顯示。
  - 日曆數據基於當前計算機時間，用戶可在設置中更改為特定服務器時區的數據。

- **遊戲公告**
  - 提供實時遊戲公告的快速瀏覽。
  - 默認數據源為國服官方公告，用戶可在設置中切換至其他數據源。

儀表盤設計旨在幫助玩家高效管理與原神相關的所有重要信息，提供便捷的功能入口和豐富的數據展示。
