# 反饋中心

![儀表盤](https://img.alicdn.com/imgextra/i4/1797064093/O1CN01yGGrhz1g6e0u3q5L6_!!1797064093.png_.webp)

反饋中心是 Snap Hutao 開發團隊為幫助用戶解決常見問題所提供的一個功能頁面，該頁面採用左右結構佈局。

## 左側功能：文檔搜索

頁面左側為內置文檔的搜索入口。用戶可通過搜索框輸入關鍵詞，快速查找到相關頁面及解決方案。

## 右側功能：信息與鏈接

頁面右側包含以下功能模塊：

- **第一部分：實用關鍵信息**

  - **設備 ID**：唯一標識當前設備，常用於反饋 Bug 或拉取錯誤日誌。
  - **設備網絡信息**：包括 IP 地址及網絡狀態。
    - 由於代理軟件實現方式的不同，部分代理可能無法被檢測到，因此“已啟用代理”情況下仍顯示“無代理”屬於正常現象。
    - Alpha 版本始終顯示歸屬地為 Oversea，這屬於預期行為。
  - **Webview2 運行時版本**：顯示工具箱運行所依賴的 Webview2 版本號。

- **第二部分：常用鏈接**

  - **GitHub Issues**：快速跳轉到 GitHub 的問題追蹤頁面，便於用戶提交問題或查看已有解決方案。
  - 訪問胡桃服務的可用性監控頁面，以瞭解服務狀態。

- **第三部分：功能指南**
  - 提供 Snap Hutao 的主要功能的文檔鏈接，包括“首頁”、“啟動遊戲”、“祈願記錄”、“成就管理”等，便於用戶快速定位文檔能入口。

## 界面優化與注意事項

- 如果反饋中心頁面中的數據未正常顯示或加載，可嘗試重啟應用或檢查網絡連接。
- 部分顯示問題可能與用戶環境（如代理設置或系統版本）相關，推薦按照提示排查具體問題。
