# 祈願記錄

抽卡是原神的核心玩法之一，無論是出於紀念、炫耀的目的，還是認真規劃抽取計劃，對歷史抽卡進行統計分析都非常重要。然而，原神官方僅提供一年的抽卡記錄，這對長期玩家來說是不夠的（~~他們卻會統計你一年中消滅了多少提米的鴿子~~）。

因此，胡桃工具箱的祈願記錄導出功能應運而生，它可以幫助玩家將祈願數據轉移到本地進行永久保存，並進行長期的數據分析。

有關原神祈願記錄導出的技術原理，請參閱[**祈願系統與導出原理**](../advanced/Gacha-system-and-export-principal.html)。

![祈願導出](https://img.alicdn.com/imgextra/i2/1797064093/O1CN01otuXYg1g6e0wnNwX2_!!1797064093.png_.webp)

如上圖所示，胡桃工具箱的祈願導出功能可以幫助用戶清晰地統計和分析抽卡數據，包括：

- 抽卡總數
- 當前已墊抽數
- 不同稀有度武器和角色的獲取數量與百分比
- 每次五星獲取的抽數及保底情況

此外，**胡桃雲用戶可以額外獲取雲端服務器計算的抽卡預測數據**。

::: note 原神抽卡記錄的延遲性
胡桃工具箱的數據來源是米哈遊服務器，因此短時間內剛抽取的祈願記錄可能無法被獲取。  
當你在遊戲內抽卡歷史記錄中看到最新記錄時，便可在胡桃工具箱中獲取對應的統計數據。
:::

## 獲取祈願數據

### SToken 刷新 <Badge text="推薦" type="tip" />

::: info 提示

- 已登錄的米遊社賬號是使用 SToken 刷新方法的前提，可參考[米遊社多賬號切換](mhy-account-switch.md)文檔在胡桃工具箱中登錄賬號。
- SToken 刷新方法暫不支持國際服。
  :::

1. 確保胡桃工具箱中已登錄需要獲取記錄的米遊社賬號。
2. 在左側菜單欄進入“祈願記錄”頁面。
3. 點擊右上角的“刷新”按鈕，選擇“SToken刷新”。
4. 等待讀取完成後，即可查看最新祈願記錄。

### 網頁緩存刷新 <Badge text="支持國際服" type="tip" />

1. 啟動原神並進入遊戲，打開遊戲內的祈願歷史記錄。
2. 在胡桃工具箱的左側菜單欄進入“祈願記錄”頁面，點擊右上角的“刷新”按鈕，選擇“網頁緩存刷新”。

::: tip 若網頁緩存刷新功能異常：

- 在程序內左下角菜單欄點擊“設置-儲存空間”。
- 執行“刪除遊戲內網頁緩存”操作。
- 重新打開遊戲中的祈願記錄界面。
- 再次嘗試刷新祈願記錄。
  :::

### 手動輸入 URL <Badge text="支持國際服" type="tip" />

1. 如果通過其他工具獲取到了祈願歷史記錄的 URL，可選擇“手動輸入 URL”並根據提示操作。
2. 請注意，URL **具有時效性**，務必儘快完成操作。

### 全量刷新模式

- 默認情況下，胡桃工具箱會在匹配到本地已有記錄後停止導入以避免重複記錄。
- 若希望完整地獲取原神服務器中所有歷史記錄，可勾選“全量刷新”，再執行刷新任務。

::: tip

- **未勾選全量刷新**不會導致祈願數據丟失。
- **勾選全量刷新**也不會導致數據重複導入。
  :::

## 導入 UIGF 祈願數據 <Badge text="UIGF" type="info" />

胡桃工具箱支持導入 `UIGF 數據格式` 的祈願記錄[^UIGF-Org]。導入步驟如下：

1. 從其他工具導出 UIGF 數據文件。
2. 在胡桃工具箱中點擊“設置-祈願記錄-遷移-導入”。
3. 在彈出的文件選擇器中選擇 `UIGF Json` 數據文件。
4. 確認數據並完成導入。

::: warning 升級舊版本 UIGF 文件
若祈願記錄版本過於老舊，可點擊“設置-祈願記錄-遷移-升級較早版本的 UIGF 文件”，完成升級後再導入。
:::

## 導出 UIGF 祈願數據 <Badge text="UIGF" type="info" />

胡桃工具箱支持將祈願記錄以 `UIGF 數據格式` 導出為 Json 文件，具體步驟如下：

1. 在左側菜單欄進入“設置”。
2. 找到“祈願記錄-遷移”部分，點擊“導出”按鈕。
3. 選擇你需要導出數據的賬號。
4. 在彈出的界面選擇文件保存路徑，點擊“導出”完成操作。

::: tip
支持導出多個賬號的祈願記錄至一個文件，無需分別操作。
:::

## 祈願記錄雲同步

> 雲同步功能基於胡桃雲賬號，需[註冊並登錄胡桃賬號](hutao-settings.md#胡桃賬號)後方可使用。

- **上傳**：在“祈願記錄”界面選擇需要同步的存檔，點擊“胡桃雲”按鈕後選擇“上傳當前存檔”。
- **下載**：點擊“胡桃雲”按鈕，選擇對應 UID 的存檔右側的下載按鈕以完成同步。
- **刪除**：點擊“胡桃雲”按鈕，選擇對應 UID 存檔右側的刪除按鈕以移除雲端數據。

[^UIGF-Org]: UIGF（統一標準化的原神數據格式）由 [UIGF-Org](https://uigf.org/) 提供。
