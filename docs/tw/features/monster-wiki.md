# 怪物資料

::: warning 已知問題
部分怪物的信息缺失，但因種種原因難以改進。與怪物資料相關的議題暫時不會被採納。
:::

![monster-wiki](https://img.alicdn.com/imgextra/i3/1797064093/O1CN01WdFqdC1g6e0sAREwO_!!1797064093.png_.webp)

怪物資料是胡桃工具箱中內置的用於查詢各個怪物的**基礎數值、各等級數值、元素抗性、掉落物品**數據的 WIKI。

## 已包含的數據

- **怪物列表**

  - 使用列表左上角的按鈕，可切換列表顯示方式：
    - “縮略圖與名稱”：顯示怪物頭像與對應名稱，便於快速瀏覽和查找。
    - “僅縮略圖”：僅顯示怪物頭像，適合已熟悉怪物外觀的玩家。
  - 數據來源於原神，部分怪物可能因數據不完全而缺少部分信息。
  - 單擊列表中的縮略圖或名稱即可跳轉至相應的怪物詳情頁面。

- **數據內容**
  - **怪物數值**
    - 怪物的基礎生命值、攻擊力、防禦力等關鍵數據。
    - 可通過界面切換怪物的等級，以查看其在不同等級的數值變化。
  - **元素抗性**
    - 包括怪物對各元素傷害的抗性百分比，用於幫助玩家優化輸出選擇。
  - **掉落物品**
    - 列出怪物可能掉落的所有物品及其掉落概率範圍，便於玩家規劃素材獲取。
