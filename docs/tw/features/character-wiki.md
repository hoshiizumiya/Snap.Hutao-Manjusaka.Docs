# 角色資料

![characters-wiki](https://img.alicdn.com/imgextra/i2/1797064093/O1CN01OKw2Tc1g6e0qbmLe0_!!1797064093.png_.webp)

角色資料是胡桃工具箱內置的綜合查詢工具，涵蓋了原神角色的基礎信息、天賦數值、命之座、培養材料等詳細數據，為玩家提供一站式信息查詢服務。

## 已包含的數據

- **角色列表**
  - 使用列表左上角的按鈕切換顯示模式：
    - “列表”模式顯示角色頭像、姓名及詳細信息。
    - “網格”模式僅顯示角色頭像與姓名。
  - 在“網格”模式中單擊角色縮略圖或姓名後，切換回“列表”視圖即可跳轉至相應角色介紹頁面。
- **數據內容**
  - **角色基本信息**
    - 包括角色基礎資料、CV 信息、屬性與培養材料。
    - 各等級下的詳細屬性數值。
- **天賦**
  - 基本介紹與各等級詳細數值：
    - 普通攻擊
    - 元素戰技
    - 元素爆發
    - 固有天賦
  - 支持切換至對應等級的天賦數值查看。
- **命之座**
- **推薦搭配**
  - 根據“胡桃數據庫”的統計信息提供的常見角色隊伍搭配、武器搭配以及聖遺物搭配推薦。
- **其他信息**
  - 包括角色立繪、名片、頭像、特色料理、衣裝信息、角色資料及背景故事。

## 將 wiki 中角色添加至養成計劃

- 在角色資料頁面，選定角色後，點擊右上角的“添加到養成計劃”按鈕，即可將該角色添加至養成計劃。
  - 默認情況下，養成計劃從角色 1 級開始（用戶可自行調整）。
  - 若需基於當前已有角色創建養成計劃，可在“[我的角色](character-data.md#將當前選定角色加入養成計算)”頁面使用養成計算功能，具體操作參見[此教程](develop-plan.md#基於玩家數據添加)。
