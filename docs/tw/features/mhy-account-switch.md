# 米遊社多賬號切換

::: tip
胡桃工具箱支持中國大陸版米哈遊論壇**米遊社**和國際版論壇 **HoYoLAB** 的賬號。HoYoLAB 默認阻攔來自中國大陸的網絡連接，Snap Hutao 軟件無法自行繞開該限制。

- 本文檔所描述的賬號切換指的是用於獲取遊戲數據的**米遊社 / HoYoLAB 賬號**，不是原神**遊戲內的賬號**
- 在使用本功能前，你需要使用官方的米遊社 App 登錄你的賬號以初始化你的米遊社功能
  - 如果在胡桃中登錄的賬號名稱格式為 “user_123456789” 則其有可能沒有被初始化

:::

![多賬號管理樣圖](https://img.alicdn.com/imgextra/i4/1797064093/O1CN01ZhnkRl1g6e0tz18y9_!!1797064093.png.png_.webp)

你可以在程序主界面的左下角，設置按鈕的上方找到賬號菜單，以管理你在胡桃工具箱中登錄的米遊社賬號。在彈出的面板中你可以添加米遊社或 HoYoLAB 賬號，登錄它們的方法是類似的：

:::: tabs

== 米遊社手機驗證碼登錄

::: warning
請確認米遊社賬號已綁定國服賬號。
:::

1. 點擊“手機驗證碼”按鈕，填寫手機號併發送驗證碼。
2. 填寫驗證碼後確認，稍等片刻即可添加賬號。

== 米遊社掃碼登錄

::: warning
請確認米遊社賬號已綁定國服賬號。
:::

1. 點擊“掃碼登錄”按鈕，等待二維碼加載。
2. 用米遊社 App 掃碼並確認登錄。
3. 稍等片刻，胡桃工具箱將添加賬號。

== HoYoLAB 密碼登錄

::: warning
請確認 HoYoLAB 賬號已綁定國際服賬號。
:::

1. 點擊“密碼登錄”按鈕，填寫賬號密碼並確認。
2. 稍等片刻，賬號將被添加。

== HoYoLAB 社媒賬號登錄

此方式依賴 WebView2 運行時組件。

::: warning
HoYoLAB 默認阻止中國大陸網絡連接。
:::

1. 點擊“三方登錄”按鈕並通過 WebView2 登錄。
2. 稍等片刻，賬號將被添加。

== Cookies 登錄

::: warning
請妥善保存 Cookie 信息，避免賬號風險。
:::

通過手動輸入 SToken 提交 Cookie 登錄。

1. 點擊目標 App 圖標選擇“手動輸入”。
2. 輸入有效 Cookie 並確認，稍等片刻賬號將被添加。
   ::::

- 完成後，您就可以在賬號管理面板切換已登錄的米遊社賬號了
  - 新添加米遊社賬號後，需在主界面左下角切換賬號的功能中，手動點擊一次新登錄的賬號，以將其設置為使用狀態
  - 你可以在賬號管理菜單中複製對應賬號的 Cookie 或從胡桃工具箱中刪除該賬號
  - 在程序沒有運行在管理員模式下時，你可以對賬號拖動排序
  - 在選中登錄賬號後，你可以
    - 點擊刷新 Cookie 按鈕來更新當前儲存的 Cookie
    - 點擊簽到按鈕來執行米哈遊論壇簽到。對於米遊社賬號，需要先解除風控狀態才可成功執行簽到。

::: info 安全提示

- `SToken` 是一個安全敏感的 Cookie 字段，你**不應該**在任何雲端或其它無法保證數據安全的設備上儲存含有 `SToken` 字段的 Cookie。
- 從胡桃工具箱中複製獲得的 Cookie 中包含該字段，請慎重考慮是否為接受 Cookie 的一方提供 `SToken` 字段。
- **在胡桃工具箱中通過密碼或 Cookie 登錄米遊社的過程中，你的數據只經米遊社服務器和本地胡桃工具箱處理，不經過包括胡桃服務器在內的任何第三方服務。**

:::

## 常見問題

### 為什麼米遊社賬號登錄狀態經常失效，添加的賬號消失

- 我們通過儲存賬號的米遊社 Cookie 來維持登錄狀態。
- 但是當用戶在瀏覽器或其它設備上**註銷賬號**後， 維持登錄狀態的 Cookie 將**失效**。
- 這會導致胡桃工具箱上的米遊社賬號在啟動後被自動移除。
- 此情況也可能因為網絡連接問題導致無法檢查 Cookie 有效性，故出現此情況後請優先重啟胡桃工具箱。
- 自 2022 年 10 月起，米遊社極大地提高了賬號被判定為有風險的概率，[賬號有風險](../advanced/exceptions.md#狀態1034-驗證失敗)時亦會令 Cookie 無法被識別為有效狀態
- 自 1.4.15 版本起，你可以通過在賬號面板中刷新 Cookie 來刷新登錄狀態
