# 軟件設置

![settings](https://img.alicdn.com/imgextra/i3/1797064093/O1CN01wDQfRa1g6e0ylR4ov_!!1797064093.png_.webp)

以下內容按照胡桃工具箱內的設置功能條目排列順序進行介紹。

## 關於胡桃

該部分包含以下信息：

- 胡桃工具箱的圖標：你可以在胡桃的[首頁](/)查看其原圖。
- [DGP-Studio](https://github.com/DGP-Studio) 已獲得相關授權使用該圖標。

## 管理員模式工具

在設置頁右側，提供管理員模式工具以供需要的用戶使用。

- **創建快捷方式**：在桌面上創建以管理員身份啟動胡桃的快捷方式。
- **以管理員身份重啟**：右鍵任務欄右下角胡桃圖標，在彈出的菜單中選擇“以管理員身份重啟”以獲取管理員權限。

## 無感驗證

- 若每日簽到和實時便箋功能觸發米哈遊的 `-1034` 錯誤，可通過該設置配置第三方驗證服務以自動通過極驗驗證。
- 配置成功後，實時便箋和簽到將自動完成驗證，避免請求失敗。

## 外觀設置

- **語言**：默認跟隨系統語言，用戶可強制設置顯示語言。
- **周起始日**：設置遊戲日曆中第一列的顯示日期。
- **背景材質**：
  - 選項包括無、亞克力、雲母（Windows 11 獨有）和變種雲母（Windows 11 獨有）。
- **顏色主題**：可切換亮色和暗色主題，默認跟隨系統設置。
- **背景圖片**：
  - 選項包括無背景圖片、本地隨機圖片、必應每日圖片、胡桃每日一圖、官方啟動器壁紙。
- **通知區域圖標**：選擇是否在通知區域顯示胡桃圖標。

## 存儲空間管理

- **數據文件夾**：
  - **打開文件夾**：打開包含用戶數據和元數據的文件夾。
  - **更換數據目錄**：更改數據文件夾存儲位置。
  - **刪除服務器轉換緩存**：清除遊戲轉服功能產生的緩存。
- **緩存文件夾**：
  - **打開文件夾**：打開包含圖片緩存的文件夾。
  - **重置圖片資源**：刪除並重新下載圖片資源。

## 快捷鍵設置

::: info 需要管理員權限
需以管理員模式啟動胡桃工具箱才可修改快捷鍵設置。
:::

- 設置鼠標自動點擊和鍵盤快捷鍵功能。
- 若快捷鍵註冊失敗（權限不足或衝突），界面底部會提示，用戶需重新設置。

## 主頁設置

- **主頁卡片**：設置是否顯示特定功能卡片（如啟動遊戲、祈願記錄）。
- **日曆服務器**：修改遊戲日曆數據源。
- **公告服務器**：更改遊戲公告數據源。

## 遊戲設置

- **刪除遊戲網頁緩存**：用於解決祈願記錄刷新異常的問題。
- **下載速度限制**：限制遊戲資源下載速度。
- **轉換服務器模式**：設置服務器轉換的執行模式。

## 祈願記錄設置

- **隱藏無記錄的歷史活動**：隱藏無記錄的祈願活動。
- **隱藏未抽取的祈願物品**：隱藏未獲取的祈願物品。
- **數據遷移**：支持 UIGF 4 格式的導入與導出。

## WebView 設置

- **分享保存模式**：
  - 選擇分享圖片時的保存方式，包括保存至剪貼板或本地。
