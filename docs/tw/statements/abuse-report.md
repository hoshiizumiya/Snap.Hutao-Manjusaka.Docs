# 濫用舉報

Snap Hutao 致力於為原神桌面端玩家提供更便捷、更豐富的非破壞性增強體驗。我們所有的功能均在不篡改、不干預遊戲服務器數據的前提下實現，不會對其他玩家的正常遊戲造成任何影響。

然而，我們**堅決反對**並將嚴肅處理以下行為：  
通過逆向修改、二次打包或注入第三方模塊的方式，**將 Snap Hutao 客戶端用於任何形式的破壞性、競爭性或作弊用途**。

## 濫用與作弊行為的定義

以下行為均被視為違反 Snap Hutao 使用規範的濫用行為，包括但不限於：

- **修改 Snap Hutao 客戶端** 以協助或加載具有破壞性或侵入性的作弊模塊；
- 使用 Snap Hutao 的注入功能或其衍生模塊，在**任何形式的競爭性、PVP 或爭鬥性社區活動中**獲取不公平優勢；
- 對遊戲內與戰鬥、判定、視覺相關的參數（如 “視野角度”、“元素爆發鏡頭” 等）進行修改後，仍參與上述競爭性活動；
- 將 Snap Hutao 注入功能用於與原項目目的不符的用途，包括但不限於：破壞他人遊戲體驗、偽造客戶端數據、或欺詐性展示遊戲效果。

> ⚠️ 注意：Snap Hutao 的“注入”功能本質上是通過內存修改實現遊戲交互的技術手段。  
> 我們僅在**單人、非競爭性**場景下提供此功能，以改善用戶體驗，不構成狹義上的“外掛”或“作弊工具”。一旦用戶將此功能用於**多人或競爭性場景**，即被視為濫用，與 Snap Hutao 官方立場無關。

## 舉報流程

當你在公開社區、論壇或視頻平臺中發現上述行為時，可通過以下方式舉報：

1. 加入 [Snap Hutao 社區](../community.md)；
2. 聯繫任意非機器人管理員；
3. 提供清晰的證據（如截圖、視頻鏈接或相關記錄）。

在證據充分的情況下，我們將對違規者關聯的遊戲 UID 執行**功能封禁**。  
被封禁的賬戶將無法使用 Snap Hutao 高級啟動器中的注入功能。
