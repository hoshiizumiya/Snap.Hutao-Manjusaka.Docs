# 第二期開發組座談會

複製自 [GitHub 討論帖](https://github.com/DGP-Studio/Snap.Hutao/discussions/280)，總結於群討論

## 現在成就導出的 json 格式也是 UIGF 的嘛

A: UIAF

## 能否適配祈願系統數據 UEFI 舊版本的導入，和 SG 一樣就可以

A: 首先它叫 UIGF 格式，其次一直是適配的，只是 SG 的導入導出規則較為寬泛，如果您從始至終僅使用且正確使用了我們的產品，是不會出現導入導出異常的問題的

## 啥時候會有 sg 那種自動檢查更新下載

A: 自動更新不能在 MSIX 包上實現，檢查更新會在後續版本推出。

## 我同一個設備上同時登錄 SG，hutao 會算作一個設備還是兩個設備

A: 兩個設備,理論上每次重啟胡桃後均相當於切換了設備

## 我感覺胡桃有些頁面的加載需要很久，會影響使用體驗，感覺胡桃的界面有點掉幀

A: 深淵統計頁面在 1.2.6 版本前確實存在加載慢的問題，已在新版本解決，如果有其他頁面的卡頓問題，通常是因為你的顯卡不夠強勁，或者胡桃通過你的集顯進行渲染。

## 屬性統計經常加載不了

A: 歷史遺留問題，可以通過備份數據後刪除 `userdata.db` 解決

## 自定義網頁功能是否還有規劃

A: 別急
