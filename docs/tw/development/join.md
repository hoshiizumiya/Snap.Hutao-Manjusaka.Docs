# 加入我們

如果你掌握本頁所述技術之一，並對加入 Snap Hutao 開發團隊以完善項目和接觸更多原神開源項目感興趣，我們熱忱歡迎你的加入！

**我們對能力沒有設定明確的門檻。探索新領域的精神、對技術的熱愛，以及負責任的態度，才是開源社區最需要的品質。**

> 如果時間有限，或者只是想解決某個具體問題，可以直接閱讀[開發手冊](contribute.md)併發起 Pull Request，我們會認真對待每一個建設性的代碼提交。
