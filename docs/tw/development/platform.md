# 胡桃開放平臺

自 2022 年 4 月 18 日在 GitHub 上創立以來，Snap Hutao 項目受到了開源社區和用戶的廣泛支持。隨著項目的發展，我們構建了一系列支持服務，作為 Snap Hutao 的中間組件。

現階段，開發組將其中一些實用服務以開放接口形式公開給開源社區，供與我們志同道合的開發者使用。我們為這些接口編寫了詳細的使用文檔，提供維護支持，並解答常見問題。無論你是經驗豐富的開發者還是初學者，我們希望這些接口能夠為你的應用開發提供幫助。

## 胡桃 API & 胡桃雲開發者許可證

胡桃 API 是公開胡桃深淵數據庫的接口服務，旨在為開源軟件開發者提供數據訪問支持。

通過申請胡桃 API 的開發者，除了獲得數據庫訪問權限外，還將額外獲得 Snap Hutao 工具箱內胡桃雲功能的完整權限作為福利。

開發者可以通過[~~胡桃 API 開放平臺~~](https://homa.snapgenshin.com/)提交申請，獲取開發者許可證。

## UIGF 可交換數據記錄

Snap Hutao 項目致力於推動用戶數據在不同開源工具間的自由流動，併為此草擬了初版[統一可交換抽卡記錄標準（UIGF）](https://uigf.org/zh/standards/UIGF.html)。

這一標準現已得到越來越多開發者的認可與採用，並在持續完善中。

眾多開發者匯聚於 [UIGF-Org](https://github.com/UIGF-org) 開源組織，我們希望通過這一平臺推動開發者與用戶的交流，獲得更多反饋與技術進步，進一步打造優秀的原神開源工具。
